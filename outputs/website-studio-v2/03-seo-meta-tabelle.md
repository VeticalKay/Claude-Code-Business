## On-Page-SEO-Checkliste pro Seite

- [ ] Genau 1 H1, danach H2/H3 hierarchisch
- [ ] Primary Keyword in H1, Title, ersten 100 Wörtern, Meta Description
- [ ] Secondary Keywords natürlich verteilt (nicht keyword-stuffen)
- [ ] Bilder mit `alt`-Tags (beschreibend, nicht keyword-spam)
- [ ] Bildnamen sprechend (`poledance-studio-salzburg.jpg`, nicht `IMG_4527.jpg`)
- [ ] Interne Verlinkung zwischen verwandten Seiten (z.B. Anfänger → Schnupperkurs → Mitgliedschaft)
- [ ] Mind. 1 Outbound Link auf vertrauenswürdige Quelle (z.B. Wellpass-Seite)
- [ ] URL kurz & sprechend
- [ ] Mobile-optimiert (Lighthouse > 90)
- [ ] Pagespeed (Core Web Vitals): LCP < 2.5s, CLS < 0.1
- [ ] HTTPS aktiv, Redirects 301 (nicht 302)
- [ ] sitemap.xml + robots.txt korrekt
- [ ] Canonical-Tag auf jeder Seite
- [ ] OpenGraph + Twitter Card Meta-Tags

## GEO/AEO — Wie KI-Engines deine Website finden

### Was KI-Engines lieben

1. **Klare, direkte Antworten** auf konkrete Fragen ("Bin ich zu alt für Poledance? Antwort: Nein, …")
2. **FAQ-Strukturen mit Schema-Markup** (siehe FAQ-Doku)
3. **Listen, Tabellen, Bulletpoints** (gut maschinenlesbar)
4. **Definitions-Sätze** ("Poledance ist ein Tanzsport, der …")
5. **Konsistente NAP-Daten** überall identisch
6. **Cross-Verlinkung** zwischen verwandten Inhalten
7. **E-E-A-T:** Experience, Expertise, Authoritativeness, Trustworthiness — Katarinas Geschichte, Methode, Erfahrung sichtbar machen

### Was du auf der Seite explizit nennst

- Wer steht hinter dem Studio? (Katarina Tolkmit, Markenstory)
- Was bietest du? (alle Kurse, Workshops, Preise)
- Wo? (Adresse, Region, Anfahrt)
- Für wen? (Zielgruppe, alle Levels)
- Wie unterschiedet ihr euch? (Methode, Ansatz, Atmosphäre)
- Was kostet es? (transparente Preise)

→ KI-Engines kombinieren diese Infos und zitieren dich, wenn jemand fragt: "Wo kann ich in Salzburg Poledance lernen?"
