Erstelle einen ausführlichen Implementierungsplan für das genannte Projekt.

Speichere den Plan in `plans/` mit dem Dateinamen `YYYY-MM-DD-projektname.md`.

Format des Plans:
```
# [Projektname]
Datum: [Datum]
Status: In Planung

## Ziel
[Was soll erreicht werden?]

## Hintergrund
[Warum wird das gemacht?]

## Schritte
- [ ] Schritt 1
- [ ] Schritt 2
- [ ] Schritt 3

## Erfolgskriterien
[Woran erkennt man, dass es fertig ist?]
```

Falls der `plans/` Ordner noch nicht existiert, erstelle ihn. Bestätige, dass der Plan gespeichert wurde.
